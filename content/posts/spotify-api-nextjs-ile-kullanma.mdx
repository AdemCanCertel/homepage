Web sitemin en sevdiğim bölümlerinden birisi [aktiviteler](/activites) yeni teknoloji API'leri denemek için de güzel bir bölüm olduğu denilebilir. Şu anda Steam, Spotify entegrasyonları vardır.

Hangi oyunu oynadığımı ve Spotify da hangi şarkıyı dinlediğimi göstermek istedim. Bunu başarabilmek için [Spotify'ın API'sini](https://developer.spotify.com/documentation/web-api/) Next.js [API Yoluyla](https://nextjs.org/docs/api-routes/introduction) entegra etmem gerekmekteydi. Bu gönderimde sizlere Spotify API ile sitenize anlık dinlediğiniz şarkıyı canlı olarak göstereceğim.

## Uygulama Oluştur

Öncelikle, Spotify API kullanmamız için kimlik oluşturmamız lazım, aşşağıdaki aşamalar ile ilerleyelim.

- [Spotfiy Geliştirme Portalına](https://developer.spotify.com/dashboard/) adresine gidin ardından Spotify hesabımız ile giriş yaptıktan sonra.
- **Uygulama Oluştur'a** tıklayalım.
- Spotify uygulamamızın adını ve açıklamasını belirleyelim, ardından oluştur'a tıklayalım.
- **Müşteri kimliğini göster**'e tıklayalım.
- **Müşteri Kimliğimiz**(Client ID) daha sonra bize lazım olucak.
- Ayarlar>Düzenle tıklayalım ve:
  - **"https://localhost:3000"** Yeniden yönlendirme URI'si olarak ekleyin.

Hepsi tamam! ✨ Artık düzgün ve yapılandırılmış bir Spotify uygulamamız var.

Kimlik doğrulama uygulamamıza bağlı olarak Spotify API ile kimlik doğrulamanın çeşitli yolları vardır. Yalnızca bir kez izin verilmesine ihtiyacımız olduğundan dolayı, "Yetkilendirme Kodu Akışını" kullanıcağız.

Öncelikle ihtiyacımız olan kapsamlar ile giriş yaparak uygulama istek yetkimizi alacağız işte URL'nin nasıl görünebileceğini dair bir örnek. Kendiniz için "client_id" ve kapsamları değiştirin.

```
https://accounts.spotify.com/authorize?client_id=e8f7664cf9a046a488360b0724804ef6
&response_type=code&redirect_uri=http
%3A%2F%2Flocalhost:3000&scope=user-read-currently-playing%20
user-top-read
```

Yetki verdikten sonra, tekrar "redirect_uri" URL'de bir "code" sorgu parametresi var. Bu değeri kaydedelim.

```
http://localhost:3000/callback?code=AQCI5..L2p03
```

Ardından, yenileme belirtecini almamız gerekecek. Daha önceki istemci kimliğini ve sırrını içeren bir **Base64** kodlu dize oluşturmamız gerekecek. [Base64 çevrimiçi kod](https://www.base64encode.org/), "client_id:client_secret" şeklinde olmalıdır.

```
curl -H "Authorization: Basic CHANGE_BASE64_HERE"
-d grant_type=authorization_code -d code=CHANGE_CODE_HERE -d redirect_uri=http%3A
%2F%2Flocalhost:3000 https://accounts.spotify.com/api/token
```

## Spotify'ın API'sini kullanma

Sonunda, bazı gerçek veriler elde ediyoruz! Next.js içinde dosyanızda üc yeni değer oluşturalım ".env.local".

```json
SPOTIFY_CLIENT_ID=
SPOTIFY_CLIENT_SECRET=
SPOTIFY_REFRESH_TOKEN=
```

> Daha önce Next'te ortam değişkenleriyle çalışmadıysanız, [okuyun.](https://nextjs.org/docs/basic-features/environment-variables)

Artık müşteri kimliğimiz ve müşteri sırrımız. "refresh_token"

```javascript
"pages/api/spotify.js"
import querystring from "querystring";

const {
  SPOTIFY_CLIENT_ID: client_id,
  SPOTIFY_CLIENT_SECRET: client_secret,
  SPOTIFY_REFRESH_TOKEN: refresh_token,
} = process.env;

const basic = Buffer.from(`${client_id}:${client_secret}`).toString("base64");
const NOW_PLAYING_ENDPOINT = `https://api.spotify.com/v1/me/player/currently-playing`;
const TOKEN_ENDPOINT = `https://accounts.spotify.com/api/token`;

const getAccessToken = async () => {
  const response = await fetch(TOKEN_ENDPOINT, {
    method: "POST",
    headers: {
      Authorization: `Basic ${basic}`,
      "Content-Type": "application/x-www-form-urlencoded",
    },
    body: querystring.stringify({
      grant_type: "refresh_token",
      refresh_token,
    }),
  });

  return response.json();
};

export const getNowPlaying = async () => {
  const { access_token } = await getAccessToken();

  return fetch(NOW_PLAYING_ENDPOINT, {
    headers: {
      Authorization: `Bearer ${access_token}`,
    },
  });
};

export default async (_, res) => {
  const response = await getNowPlaying();

  if (response.status === 204 || response.status > 400) {
    return res.status(200).json({ isPlaying: false });
  }

  const song = await response.json();
  const isPlaying = song.is_playing;
  const title = song.item.name;
  const artist = song.item.artists.map((_artist) => _artist.name).join(", ");
  const album = song.item.album.name;
  const albumImageUrl = song.item.album.images[0].url;
  const songUrl = song.item.external_urls.spotify;

  return res.status(200).json({
    album,
    albumImageUrl,
    artist,
    isPlaying,
    songUrl,
    title,
  });
};
```

```javascript
"components/spotify.js"
{ NextSeo } from 'next-seo';
import useSWR from 'swr';
import { SiSpotify } from 'react-icons/si';

export default function Home() {
    const fetcher = (url) => fetch(url).then((r) => r.json());
    const { data } = useSWR('./pages/api/spotify', fetcher);
    return (
    <div>
        <NextSeo />
            <section className='full-container'>
                <a
                 target='_blank'
                 rel='noopener noreferer'
                 className='relative flex items-center pb-5 space-x-4 rounded-md'
                >
                <div className='w-16'>
                  {data?.isPlaying ? (
                    <img
                     className='w-16'
                     src={data?.albumImageUrl}
                     alt={data?.album}
                    />
                  ) : (
                    <SiSpotify size={60} color={'#1ED760'} />
                  )}
                </div>
                    <div>
                        <p className='text-xl leading-7 dark:text-white text-black'>
                           {data?.isPlaying ? data.artist : 'Spotify'}
                        </p>
                        <p className='text-zinc-500 dark:text-zinc-50'>
                           {data?.isPlaying ? data.title : 'Not Listening'}
                        </p>
                    </div>
                </a>
            </section>
    </div>
    );
}
```
